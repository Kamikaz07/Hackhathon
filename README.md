# Stick Brawl

A multiplayer 2D fighting game where stick figures battle in dynamic arenas.

## Setup

1. Install Python 3.x
2. Install dependencies:
```bash
pip install -r requirements.txt
```

## Running the Game

1. Start the server:
```bash
python server.py
```

2. Start the client(s):
```bash
python client.py
```

## Controls

- Arrow keys: Move left/right
- Space: Jump
- E: Attack/Use weapon

## Features

- Multiplayer support for up to 4 players
- Dynamic arenas with platforms and obstacles
- Weapon pickups and physics-based combat
- Last man standing gameplay mode
